# 싸피인간 코딩장비 공유 서비스 - COALA
> 영화에 대한 커뮤니티와 위시리스트, 검색을 통해 영화 추천 기능을 갖춘 사이트

<br>

### 프로젝트 기간 🗓️
- 2023.07.04 ~ 2023.08.18

<br>

### 개발환경 🛠️
>
> Java 17.
> >
> Spring Boot 3.1.2.
> > 
> Node.js 18.
> >
> Next 13.2.4.
> >
> React 18.2.0.
